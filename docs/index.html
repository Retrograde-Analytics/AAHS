Dashboard

This dashboard contains an interactive map that shows the locations of and general information about African American historical sites in Philadelphia. The dataset that the map is built on was created and maintained by the Preservation Alliance Whose mission is to actively promote the appreciation, protection, appropriate use, and development of the Philadelphia regions historic buildings, communities, and landscapes.

The original dataset can be viewed on the second page of this dashboard or here.

It was accessed by the author of this dashboard through Open Data Philly.
